# simple-math-web

basic flask app to solve linear and quadratic equations.  
no styling frameworks used.  
just html, css, flask.  

## usage

clone the repo  
install requirements  
run `app.py`

## structure

- `index.html` - homepage  
- `lin.html` - linear equations (2 variables)  
- `quad.html` - quadratic equation solver  
- `app.py` - flask app
